#  Gaziza’s Shop — Online Store for Women

**Author:** Gaziza Bakir  
**Project Type:** Midterm Project — Web Technologies  


---

##  Description
**Gaziza’s Shop** is a modern and elegant online store designed for women’s fashion.  
It features a minimalistic and stylish interface using **baby pink, white, and black** as the main color palette.  
The website demonstrates responsive web design principles with **HTML, CSS, Flexbox, Grid, and Bootstrap 5**.

---

##  Pages
The website includes **5 pages**:
1. **Home** — Hero section with welcome message and featured products.  
2. **About** — Short introduction about Gaziza’s Shop and its mission.  
3. **Shop** — Product cards with prices and hover effects.  
4. **Gallery** — Image showcase with smooth hover scaling.  
5. **Contact** — Contact form and table with size/price info.

---

##  Features Implemented
- Fully **responsive layout** (desktop, tablet, mobile)
- **Sticky navigation bar** with hover animation
- **Gradient background** (baby pink → white)
- **Flexbox** for header and hero layout
- **CSS Grid** for product cards and gallery
- **Hover & focus effects** on buttons and links
- **Form** for customer messages
- **Table** for product/sizing comparison
- **CSS variables**, **pseudo-classes**, and **media queries**
- **Lazy loading** for images
- **Google Fonts** integration (Poppins & Inter)

---

##  Technologies Used
- **HTML** (semantic structure)
- **CSS** (Flexbox, Grid, variables)
- **Bootstrap 5**
- **Google Fonts**
- **GitHub Pages** (for hosting)

---

##  Responsive Design
The layout automatically adapts for:
- Large screens (3-column product grid)
- Tablets (2-column grid)
- Mobile devices (single column + stacked header)
